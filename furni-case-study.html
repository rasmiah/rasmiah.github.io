Furni — Furniture Website

A calm, minimal furniture shopping experience with soft colors, clear hierarchy, and gentle interactions that make browsing feel easy.

Role: UI/UX Designer

Tools: Figma

Platform: Web

Explore in Figma

  • Prototype: Open interactive prototype
  • Design file: View Figma design file

Problem

Many furniture websites feel cluttered, making it hard to focus on the products. I wanted to create a calmer browsing experience with clear hierarchy and fewer distractions.

Goal

Design a simple, modern flow where users can move from home to shop to product to cart quickly and without confusion.

Target User

People decorating their homes who prefer a clean, minimal shopping experience with soft colors, clear typography, and straightforward navigation.

Key Interactions

Each short clip focuses on one part of the experience: how the page moves, how the user navigates, and how small interactions support the flow.

Home Page Scroll

This clip shows the home page hero, featured content, and the first sections of the site. The scroll is calm on purpose: I wanted the user to feel guided, not rushed.

Navigation Between Sections

Here, the user moves between the home page, shop, and other sections. The navigation is kept simple, with consistent spacing, clear labels, and smooth transitions so it feels easy to jump around.

Shopping Card & Add to Cart

This interaction focuses on the product cards. When the user hovers, the background changes and a plus icon appears. This small movement invites the user to interact without adding visual noise.

Cart Quantity & Removal

In the cart, the user can increase or decrease quantities with plus and minus buttons, or remove an item completely with the X icon. This makes the cart feel more like a real product and less like a static design.

Design Decisions

Visual Style

I used a soft palette of deep green , muted greens , warm neutrals, and a yellow accent , paired with Inter in multiple weights to keep the interface calm, readable, and product-focused.

Components

I built reusable components for navigation, buttons, inputs, product cards, feature sections, testimonials, and the cart. This keeps the layout consistent and makes changes faster.

Micro-interactions

Subtle hover states, plus icons on product cards, and responsive cart controls add feedback and life to the UI without creating visual noise.
